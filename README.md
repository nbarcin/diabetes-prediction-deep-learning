# 🧠 Diabetes Prediction with Deep Learning (Pima Dataset)

This project demonstrates how to build a deep learning model to predict diabetes using the Pima Indians Diabetes dataset.

---

## 📊 Project Overview

Diabetes prediction is a critical problem in healthcare analytics. In this project, we apply a deep learning approach to classify whether a patient is diabetic based on medical features such as glucose level, BMI, age, and blood pressure.

We build a fully connected neural network using TensorFlow/Keras to learn complex patterns from structured (tabular) data.

---

## 📁 Dataset

The dataset used is the **Pima Indians Diabetes Dataset**, which contains:

- 768 samples
- 8 input features
- 1 target variable (`Outcome` → 0: Non-diabetic, 1: Diabetic)

### 🔢 Features:
- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age
